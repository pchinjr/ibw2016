Create & Collaborate

From Oct 7 to Oct 11, use technology to show off your love for babywearing. You can make an educational site or an app idea to help babywearers around the world become more connected and empowered. Work with other people on your ideas and see how far you can get in a couple of days.

Potential Ideas

  • Create a Blog
  • Make a website about a specific babywearing issue
  • Make a webapp that will sort your wraps by color and size
  • Submit an edit to a babywearing related wikipedia
  • Create an online photo gallery of your babywearing photos

Resources

I started this idea after I created Random Carry during the 2015 IBW. The idea came from someone else and I just ran with it. At the time I had just started to teach myself JavaScript and was an awesome challenge. This year, I want to see everyone at any skill level participate!

If you've never made a website before, I suggest using www.squarespace.com They offer a one month free trial and is really easy to use.

For those looking to get started in learning how to code I suggest Free Code Camp. We have an awesome local community in Hampton Roads and its FREE.

You can submit your finished sites to the Facebook Group or submit a link as a pull request to this github in the README
